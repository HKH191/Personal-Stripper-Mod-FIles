to add addon peds add the name of the peds hash then a comma (,) then the a name for the ped in 'Personal Stripper.ini' file 
a list of peds and pictures can be found here : https://wiki.rage.mp/index.php?title=Peds

to add the ped open 'Personal Stripper.ini' file inside the Personal Stripper Folder
for example if i wanted to add Abigail i would write :
ig_abigail (The Peds Spawn Hash) then comma (,)  and lastly a name, Abigail
so you should have this :
ig_abigail,Abigail

to add New locations
Open the Personal Stripper Folder and read the file "how to add a new loccation to Personal Stripper"
